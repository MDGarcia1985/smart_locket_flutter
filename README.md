About this app
=================

Smart Locket Flutter is a mobile application for managing a Bluetooth-enabled smart locket device.

## Features

- **Firebase Authentication**: Production sign-in with email/password, Google, and Apple
- **Bluetooth Connectivity**: Connect to smart locket devices via BLE
- **Photo Management**: Upload and manage photos on the device
- **Device Gallery**: View photos stored on the connected locket
- **Cross-Platform**: Supports Android, iOS, Windows, macOS, Linux, and Web

## Firebase setup (required for auth)

Auth is powered by Firebase. Before running the app, configure your Firebase project and generate `lib/firebase_options.dart`. See **[FIREBASE_SETUP.md](FIREBASE_SETUP.md)** for step-by-step instructions.
